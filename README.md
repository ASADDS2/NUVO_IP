# NUVO Pool Service

This microservice is responsible for managing pools and investment logic within the NUVO ecosystem.

## Features

- Pool management (creation, updates, retrieval)
- Investment tracking and processing
- Integration with other NUVO services

## Getting Started

### Prerequisites

- Java 17 or higher
- Maven
- PostgreSQL

### Build

```bash
./mvnw clean install
```

### Run

```bash
./mvnw spring-boot:run
```
